### Hello Name
It's just simple package for print say hi

### Get Started

```
npm install --save-dev @raihanmuhammad/hello-name
```
if you using yarn @raihanmuhammad/hello-name
```
yarn add -D 
```

### How to use

```
import { sayHi } from '@raihanmuhammad/hello-name';

const name = sayHi("raihan");
// console.log(name);
// alert(name);
```
